CSS


es el lenguaje de estilos utilizado para describir la presentación de documentos HTML o XML (incluyendo varios lenguajes basados en XML como SVG, MathML o XHTML). CSS describe cómo debe ser renderizado el elemento estructurado en la pantalla, en papel, en el habla o en otros medios.

Atributos

Nombre del Atributo Descripción Valores Posibles Ejemplo de Uso Imagen Representativa Fuente de Información
font-size La propiedad font-size especifica la dimensión de la letra. Este tamaño puede, a su vez, alterar el aspecto de alguna otra cosa, ya que se usa para calcular la longitud de las unidades em y ex.
  • Valores absolutos: xx-small, x-small, small, medium, large, x-large, xx-large
  • Valores relativos: larger, smaller
  • Unidades de longitud
  • Porcentajes (en relación al elemento padre)
  • REM (Root Em), una unidad relativa al elemento raíz HTML
Fuentes
border-radius La propiedad border-radius CSS redondea las esquinas del borde exterior de un elemento. Puedes establecer un único radio para crear esquinas circulares o dos radios para crear esquinas elípticas. son uno, dos, tres o cuatro valores. Se pueden especificar en unidades de longitud CSS válidas, como px, rem, em, ch, vh, vw, entre otras. También se pueden especificar en porcentajes.
Fuentes
display la propiedad display establece los tipos de visualización interna y externa de un elemento. La tipo externa establece la participacion de un elemento en flow layout; la tipo interna establece el layout(Diseño) de los hijos. Los elementos de tipo div, encabezados, formularios, secciones o párrafos son elementos que tienen por defecto el valor block. Por otro lado, los elementos como span, cursiva, negrita, enlace e imagen tienen por defecto el valor inline
Fuentes
flexbox fue diseñado como un modelo unidimensional de layout, y como un método que pueda ayudar a distribuir el espacio entre los ítems de una interfaz y mejorar las capacidades de alineación. Este artículo hace un repaso de las principales características de flexbox, las que exploraremos con mayor detalle en el resto de estas guías
  • row: Los elementos se alinean de izquierda a derecha, como palabras en una oración
  • row-reverse: Similar a row, pero la alineación va de derecha a izquierda
  • column: Los elementos se apilan verticalmente, de arriba a abajo
  • column-reverse: Similar a column, pero la pila va de abajo hacia arriba
Fuentes
position La propiedad position de CSS especifica cómo un elemento es posicionado en el documento. Las propiedades top, right, bottom, y left determinan la ubicación final de los elementos posicionados.
  • Static: (estático)
  • Relative: (relativo)
  • Absolute: (absoluto)
  • Fixed: (fijo)
  • Sticky: (pegadizo)
